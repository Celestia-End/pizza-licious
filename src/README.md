## App Features
- Custom Pizzas: Choose size (Small, Medium, Large), crust, sauces, cheeses, meats, and toppings. Auto-calculated total. Items can include + extras.
- Drinks: Add different sized drinks (Small, Medium, Large).
- Garlic Knots: Add garlic knots.

## How to Use
1. Select from the main menu:
    - Create a pizza
    - Add a drink
    - Add garlic knots
    - View order
    - Checkout
2. Customize items using number-based selections.
3. Checkout displays the total for all items.
